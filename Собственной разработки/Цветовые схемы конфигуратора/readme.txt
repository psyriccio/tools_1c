Настройки цветовой схемы хранятся в файле 
c:\Documents and Settings\[ИмяПользователя]\Application Data\1C\1Cv82\1cv8.pfl

Для установки цветового оформления нужно заменить этот файл одним из файлов настроек. 
ВНИМАНИЕ: помимо цветового оформления поменяются и другие настройки конфигуратора. 
Для изменения ТОЛЬКО цветового оформления нужно править указанный файл вручную.
